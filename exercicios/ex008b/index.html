Código-Fonte e Pré-Formatação

O comando document.getElementById('test') é escrito em linguagem JavaScript.
Obs: Escrito da forma acima fica muito feio porque a fonte das letras não é mono espaçada gerando até confusão na leitura do comando e identificação de que se trata de um comando de um programa.

agora veja como esse mesmo comando fica usando a tag <code>: document.getElementById('test')

Outro exemplo com Phyton:
num = int(input('Digite um número')) if num % 2 == 0: print(f'0 npumero {num} é par') else: print(f'0 número {num} é impar') print('FIM DO PROGRAMA')

No exemplo acima você pode notar que apesar de estar com monoespaçamento, ainda não está com uma cara legal para mostrar que se trata de um comando phyton! Para resolver isso, usaremos a tag <pre> em conjunto com a tag <code>. Veja como ficou o mesmo comando phyton agora:


num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'0 npumero {num} é par')
else:
    print(f'0 número {num} é impar')
print('FIM DO PROGRAMA')
