Cores em CSS através do codigo Inline

Para entender esse exercicío, olhe bem atentamente o código fonte do arquivo digitado dentro do parâmetro style em cada um dos 4 titulos abaixo. As cores de todos os titulos são exatamente as mesmas. Mas o formato de representação muda conforme o padrão escolhido: por Nome, por código Hexadecimal, por código RGB ou por código HSL)

Este título está colorido usando cores por nomes (como vinhamos fazendo até o momento)

No titulo acima, o fundo azul ficou com NOME configurado da seguinte forma: darkblue. Ou seja, o nome simplesmente escolhido na lista do VBS.

Já, o texto acima ficou com NOME configurado da seguinte forma: white. Ou seja, o nome simplesmente escolhido na lista do VBS.

Este título está colorido usando cores por código Hexadecimal (3 pares de códigos alfanuméricos entre 0 a 9 e entre A a F: vermelho, verde e azul)

No titulo acima, o fundo azul ficou com HEXADECIMAL configurado da seguinte forma: #00008b. Ou seja, 00 de vermelho, 00 de verde e 8b de azul.

Já, o texto acima ficou com HEXADECIMAL configurado da seguinte forma: #ffffff. Ou seja, ff de vermelho, ff de verde e ff de azul.

Este título está colorido usando cores por código RGB (3 blocos de códigos numéricos entre 0 a 255: vermelho, verde e azul)

No titulo acima, o fundo azul ficou com RGB configurado da seguinte forma: rgb(0, 0, 139). Ou seja, 0 de vermelho, 0 de verde e 139 de azul.

Já, o texto acima ficou com RGB configurado da seguinte forma: rgb(255, 255, 255). Ou seja, 255 de vermelho, 255 de verde e 255 de azul.

Este título está colorido usando cores por código HSL (3 blocos de códigos numéricos entre 0 a 255, 0% a 100% e 0% a 100%: matiz/hue, saturação/saturation e iluminância/lightness)

No titulo acima, o fundo azul ficou com HSL configurado da seguinte forma: hsl(240, 100%, 27%). Ou seja, 240 de matiz, 100% de saturação e 27% de iluminância.

Já, o texto acima ficou com HSL configurado da seguinte forma: hsl(0, 0%, 100%). Ou seja, 0 de matiz, 0% de saturação e 100% de iluminância.

Este título está colorido usando cores por código RGBA (3 blocos de códigos numéricos entre 0 a 255: vermelho, verde e azul) + (1 código numérico entre 0.000 a 0.999: grau de transparência)

No titulo acima, o fundo azul sofreu aplicação de 50% de transparência, acrescido posteriormente aos códigos RGB que já vimos acima: rgba(0, 0, 139, 0,5). Ou seja, 0 de vermelho, 0 de verde, 139 de azul e 0,5 de transparência.

Já, o texto acima ficou mantido sem acrécimo de transparência com RGB configurado da seguinte forma: rgb(255, 255, 255). Ou seja, 255 de vermelho, 255 de verde e 255 de azul.

Este título está colorido usando cores por código HSLA (3 blocos de códigos numéricos entre 0 a 255, 0% a 100% e 0% a 100%: matiz/hue, saturação/saturation e iluminância/lightness) + (1 código numérico entre 0.000 a 0.999: grau de transparência)

No titulo acima, o fundo azul sofreu aplicação de 50% de transparência, acrescido posteriormente aos códigos HSL que já vimos acima: hsla(240, 100%, 27%, 0,5). Ou seja, 240 de matiz, 100% de saturação, 27% de iluminância e 0,5 de transparência.

Já, o texto acima ficou mantido sem acréscimo de transparência com RGB configurado da seguinte forma: rgb(255, 255, 255). Ou seja, 255 de vermelho, 255 de verde e 255 de azul.
