Cores Degradee em CSS através do código Internal (local)

Para entender esse exercicío, olhe bem atentamente o código fonte CSS do arquivo digitado dentro da tag <style>, que foi inserido dentro da área do <head> com uso de código CSS (internal/local/CSS Style). Esse tipo de CSS permite extender a configuração de cores a toda essa página se indicarmos que ela deve ser aplicada a todo a área do <body>, através do seletor "body {}". Essa página foi criada com a seguinte formatação do codigo css:
<style>
body {
background-image:linear-gradient(135deg, #3B61CE, #DE84D7, #FF4B5A, #FF873F, #FFBA44);
}
