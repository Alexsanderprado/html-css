Fontes Externas Baixadas

O titulo H1 acima foi configurado com uma fonte externa baixada no site DAFONT. Esta fonte é no formato ".ttf" (opentype). Já esse paragráfo foi confifurado com uma fonte sem serifa escolhida aletaoriamente pelo navegador.

Tanto o paragráfo quanto o titulo, receberam as configurações de suas fontes no método CSS Internal. As propriedades foram inseridas dentro da área <head> do documento HTML desenvolvido com visual Basic, através da tag <style>.
